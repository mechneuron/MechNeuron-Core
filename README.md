# MechNeuron Core Architecture

We do not wire components. We enforce mathematics onto silicon.

MechNeuron is an evolving, deterministic hardware-to-software framework. It abandons heavy, abstracted machine learning libraries in favor of raw, mathematically pure forward-pass neural networks written entirely from scratch in NumPy. The visual identity and structural geometry of this repository strictly dictate symmetrical, highly structured architecture with zero organic or biological resemblances. 

This repository houses the cumulative engineering pipeline, progressing from localized desk-scale matrices to floor-scale spatial computation. 

---

## 📐 The Three Pillars of Execution

1. **Raw Computation:** Microcontrollers (Arduino/ESP32) are strictly utilized as passive display matrices and zero-latency data conduits. They execute zero neural math. The spatial and latent calculations are handled entirely in Python.
2. **Cinematic Smoothing:** All Python pipelines enforce an Exponential Moving Average (EMA) algorithm. Data is transmitted to the physical hardware with sub-millisecond fluidity, completely eliminating visual banding and jitter at 60fps.
3. **Unbreakable Telemetry:** The serial bridge relies on strict packet-framing (header-byte synchronization) and bidirectional acknowledgment payloads to prevent buffer overflows and latency degradation.

## 📂 System Evolution

### Series 1: The Breathing Matrix
**Directory:** `/Series_1`
The foundational hardware-to-software conduit. An untrained, deterministic neural autoencoder that ingests continuous time-series orbital functions and projects them into a highly expanded 18-channel dimensional output.
* **Software:** Variance-amplified NumPy network with dynamic range expansion.
* **Hardware:** Arduino Uno utilizing Direct Port Manipulation for high-speed software PWM.
* **Key Achievement:** Perfecting the unbreakable 64-byte serial buffer limit via acknowledgment handshakes.

### Series 2: The Tactile Override
**Directory:** `/Series_2`
The establishment of a zero-latency sensory bridge. Transforming the one-way matrix into a bidirectional, closed-loop system.
* **Software:** Input vector restructuring. A dedicated dual-filter EMA sanitizes incoming raw electromagnetic noise before it enters the neural layer. Deterministic weights force absolute, symmetrical authority for both physical inputs.
* **Hardware:** Integration of dual B10K linear potentiometers operating on a shared ground, physically steering the latent space in real-time.
* **Key Achievement:** Sub-millisecond tactile intercept and 15-channel matrix payload generation.
